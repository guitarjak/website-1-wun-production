❌ ไม่มีพื้นที่ของตัวเอง

Website 1 วัน Challenge

เรียนรู้ทีละโมดูล ตั้งแต่ Mindset จนขึ้นออนไลน์พร้อมโดเมน

Module 1: Mindset & Overview

• ทำไมเราต้องมีเว็บไซต์? (โดยเฉพาะคนธรรมดา)
• การทำเว็บไซต์สมัยก่อน (และในสมัยนี้)
• สิ่งที่เราจะสร้างกัน (ฉบับรวบรัด)
• ภาพรวมของเครื่องมือที่ต้องใช้

Module 2: Setting up our playground

• การลง Node.js / Gemini CLI / Qwen CLI
• การใช้ Terminal หรือ Command Prompt เบื้องต้น
• การสร้าง Website "Hello World" ของเรา!

Module 3: Customizing locally

• โครงสร้างของไฟล์เมื่อสร้าง Website (แบบง่ายๆ)
• การใช้เครื่องมือช่วยออกแบบ (Stitch)
• การแก้ไข ภาพ ข้อความ ลิงก์ หน้าต่างๆ

Module 4: Hosting with Github Pages

• สร้างรหัส Github ของเรากัน
• การติดตั้ง Github Desktop
• การสร้าง Repo ของเรา
• เอาไปโชว์ชาวบ้านกันเถ๊อะะ

Module 5: Custom Domain

• โดเมนคืออะไร (ฉบับเด็ก 5 ขวด)
• การซื้อโดเมน (Namecheap / Hostinger)
• การเชื่อมต่อ DNS กับ Github Page
• อย่าลืมเปิด SSH ด้วยละ

Module 6: Bonus

• การทำให้ดีไซน์คงที่ด้วย guideline.html
• การเพิ่ม Form เพื่อรับข้อมูล
• การตั้งชื่อไฟล์เบื้องต้น
• การสำรองข้อมูลเว็บไซต์ของเรา
